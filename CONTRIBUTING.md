# 贡献指南

## 欢迎贡献

感谢你对 Edict 项目的兴趣！

## 如何贡献

### 1. Fork 项目
点击 GitHub 页面右上角的 Fork 按钮

### 2. 创建分支
```bash
git checkout -b feature/your-feature
```

### 3. 开发
- 遵循现有代码风格
- 添加测试（如适用）
- 更新文档

### 4. 提交
```bash
git commit -m "feat: 添加新功能"
git push origin feature/your-feature
```

### 5. 创建 Pull Request
在 GitHub 上创建 PR，描述你的修改

## 代码规范

### Python
- 使用 type hints
- 遵循 PEP 8
- 使用 black 格式化

### Git Commit Message
```
feat: 新功能
fix: 修复bug
docs: 文档更新
refactor: 代码重构
test: 测试
```

## 问题反馈

通过 GitHub Issues 报告bug或提出功能请求。
