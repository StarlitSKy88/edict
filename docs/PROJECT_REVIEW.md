## 一、项目定位

### 1.1 核心定位

Nick 是一个**企业级多 Agent 协作系统**，采用中国古代「三省六部」官僚制度设计，实现 AI Agent 之间的有序协作与任务流转。

### 1.2 面向人群

| 人群 | 适用场景 |
|------|---------|
| **技术团队** | 软件开发团队的项目管理、代码审查 |
| **创业公司** | 小团队的任务分派与协作 |
| **个人开发者** | 多任务管理、AI辅助编程 |
| **企业** | 内部流程自动化、Agent 协作 |

### 1.3 核心价值

1. **分权制衡** - 类似三省六部的审核机制，防止 AI 盲目执行
2. **可观测** - 实时看板，任务流转全程可见
3. **可干预** - 人工可随时叫停、修改、恢复任务
4. **企业级** - 熔断器、分布式锁、审计日志、成本分析
