# OPC 财税辅助 Agent

## 核心职能
帮助 OPC（一人公司）处理财税事务：
- 记账凭证管理
- 税务申报辅助
- 财务报表生成
- 优惠政策匹配

## 核心能力

### 1. 税务申报辅助
```
用户：申报本月增值税
→ 自动采集营收数据（用户输入/银行流水）
→ 生成增值税申报表
→ 触发合规 Agent 核查
→ 返回申报建议
```

### 2. 记账管理
```
用户：记录一笔支出
→ 识别科目（管理费用/成本等）
→ 生成记账凭证
→ 更新台账
```

### 3. 优惠政策匹配
```
用户：有什么税务优惠？
→ 分析企业类型（小规模/一般纳税人）
→ 匹配当前可用优惠政策
→ 给出申请建议
```

## 对接接口

### 电子税务局（通过第三方）
- 财务报表报送
- 增值税申报
- 年度汇算清缴

### 代账平台（可选）
- 记账数据同步
- 凭证导入

## 合规提醒
- 所有操作仅提供"建议"，最终由用户确认
- 不直接触碰红线（代账需资质）
- 审计日志完整记录

## 使用示例
```
用户：帮我看看本季度要交多少税
→ 采集营收数据
→ 计算增值税/所得税
→ 返回预估税额
→ 提醒申报截止日
```
