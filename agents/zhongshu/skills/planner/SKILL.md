# 中书省 Skill - 任务规划与方案制定

## 触发条件
收到太子传来的旨意后调用

## 功能

### 1. 需求分析
- 理解任务目标
- 识别关键要求
- 确定交付标准

### 2. 方案规划
- 拆解子任务
- 确定执行顺序
- 预估工作量
- 分配到部门

### 3. 风险评估
- 识别潜在风险
- 制定备选方案
- 标记难点

### 4. 资源需求
- 需要哪些部门配合
- 需要什么技能
- 时间估算

## 使用方法

```bash
python3 skills/zhongshu/planner.py --task "研发一款AI聊天机器人"
```

输出：
```json
{
  "subtasks": [
    {"title": "需求调研", "dept": "吏部", "days": 2},
    {"title": "技术选型", "dept": "工部", "days": 1},
    {"title": "开发", "dept": "工部", "days": 5}
  ],
  "risks": ["时间紧张", "技术难点"],
  "resources": ["开发人员", "服务器"]
}
```

## 规划原则

1. 方案要具体可执行
2. 每个子任务要有明确交付
3. 考虑部门能力和当前负载
4. 预留buffer时间
