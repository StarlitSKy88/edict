# 教皇 Skill - 消息分拣与任务识别

## 触发条件
当收到用户消息时自动调用

## 功能

### 1. 消息分类
- 闲聊/问答 → 直接回复
- 正式旨意 → 创建任务
- 复杂任务 → 转交红衣主教团

### 2. 意图识别
识别消息意图：
- 信息查询
- 任务请求
- 闲聊
- 确认

### 3. 任务提取
从旨意中提取：
- 任务目标
- 交付物
- 优先级
- 截止时间

### 4. 紧急度判断
- 紧急 → 标记加急
- 普通 → 正常排队

## 使用方法

```python
# 消息分拣
python3 skills/pope/classifier.py --message "帮我想一个营销方案"

# 输出
{
  "type": "task",
  "intent": "创建营销方案",
  "priority": "normal",
  "target": "营销方案",
  "deliverable": "文档"
}
```

## 关键规则

1. 宁可少建任务，不可把闲聊当旨意
2. 标题必须简洁（10-30字）
3. 禁止在标题中出现文件路径、URL
4. 提取核心动词作为任务类型
